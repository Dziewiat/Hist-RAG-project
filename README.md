
# Installation Requirements:
1. System Dependecies 
```
sudo apt install git-lfs
```
2. Python Packages 
```
git lfs pull
```
3. Hugging Face Authentication
```
hf auth login
```

4. Create Required Directiores
```
mkdir -p retrieval/output
```
